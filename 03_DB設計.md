1. DB設計・モデル関係
1-1. User
カラム名	型	備考
id	integer	PK
name	string	ユーザー名
email	string	ユニーク、ログイン用
password_digest	string	Devise or has_secure_password
created_at	datetime	
updated_at	datetime	
1-2. Category
カラム名	型	備考
id	integer	PK
user_id	integer	FK: User
name	string	カテゴリ名
color	string	任意: タイマーやグラフ色
created_at	datetime	
updated_at	datetime	

関係:

User has_many :categories

Category belongs_to :user

Category has_many :tasks

1-3. Task（作業履歴）
カラム名	型	備考
id	integer	PK
user_id	integer	FK: User
category_id	integer	FK: Category（未分類はデフォルト）
name	string	作業名
start_time	datetime	作業開始時間
end_time	datetime	作業終了時間
duration	integer	秒数 or 分数
pomodoro_sets	integer	何セット消化したか（ポモドーロモード用）
created_at	datetime	
updated_at	datetime	

関係:

User has_many :tasks

Task belongs_to :user

Task belongs_to :category

1-4. TimerSetting
カラム名	型	備考
id	integer	PK
user_id	integer	FK: User
work_duration	integer	分数、デフォルト25分
break_duration	integer	分数、デフォルト5分
long_break_duration	integer	分数、オプションで長休憩
sets	integer	デフォルト4
created_at	datetime	
updated_at	datetime	

関係:

User has_one :timer_setting
